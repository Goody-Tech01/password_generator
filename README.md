# password_generator
A modern desktop application built with python and customtkinter that generates strong, secure, and customizable passwords. Users cane choose the desired password length and select whether to include uppercase letters,lowercase letters, and special characters. The application provides an intuitive interface,making password creation quick,simple, and secure

## Features 
-Generate secure password
-Choose password length
-include letters,numbers and symbols
-copy password to clipboard

## Technologies used 
-python
-CustomTkinter
-Tkinter messagebox
